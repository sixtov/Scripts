Collect Indicators for selected cases

Divide each indicator by corresponding load in Kg
this results in a per unit load indicator
The indicators consist of a strain-based and a displacement-based
load indicators for both the left and right sides
st0500Indicators = [st0500.l.sPindicator ...
                    st0500.r.sPindicator ...
                    st0500.l.dPindicator ...
                    st0500.r.dPindicator]/0.5;
st1000Indicators = [st1000.l.sPindicator ...
                    st1000.r.sPindicator ...
                    st1000.l.dPindicator ...
                    st1000.r.dPindicator]/1.0;
st1500Indicators = [st1500.l.sPindicator ...
                    st1500.r.sPindicator ...
                    st1500.l.dPindicator ...
                    st1500.r.dPindicator]/1.5;
st2000Indicators = [st2000.l.sPindicator ...
                    st2000.r.sPindicator ...
                    st2000.l.dPindicator ...
                    st2000.r.dPindicator]/2.0;
st2500Indicators = [st2500.l.sPindicator ...
                    st2500.r.sPindicator ...
                    st2500.l.dPindicator ...
                    st2500.r.dPindicator]/2.5;
st3000Indicators = [st3000.l.sPindicator ...
                    st3000.r.sPindicator ...
                    st3000.l.dPindicator ...
                    st3000.r.dPindicator]/3.0;

Combine the Indicators

Group all indicators to determine basic statistics such as
average and standard deviation
indicators       = [st0500Indicators; ...
                    st1000Indicators; ...
                    st1500Indicators; ...
                    st2000Indicators; ...
                    st2500Indicators; ...
                    st3000Indicators];

Show Results

Display the average values and the standard deviations
disp(' ')
disp(sprintf('%%Mean Indicators\navgIndicator = [%s];',sprintf('%f; ',mean(indicators))))
disp(sprintf('%%Stdv Indicators\nstdIndicator = [%s];',sprintf('%f; ', std(indicators))))
disp(' ')
